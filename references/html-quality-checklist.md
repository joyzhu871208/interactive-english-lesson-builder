# HTML 自检清单

生成 HTML 后逐项检查，并在交付前修复发现的问题。

## 教学匹配

- 互动任务与已确认教学目标直接对应。
- 页面用语、题目难度和反馈适合目标学段。
- 活动流程与已确认方案一致。
- 没有为了趣味加入无教学价值的步骤。

## 功能

- HTML 可在浏览器直接打开。
- 所有按钮均有效。
- 正确、错误、完成等状态反馈清晰。
- 重置、重新开始或下一步操作有效。
- 不存在 JavaScript 控制台报错。
- 不存在无效占位符、TODO 或空白题目。

## 投屏与视觉

- 主要文本、题目和按钮在大屏上清晰可读。
- 页面层级明确，重点内容突出。
- 内容不过度拥挤，必要时合理分页或分步显示。
- 色彩对比充足，反馈不只依赖颜色。
- 动效简短、适度，不干扰教学节奏。

## 可编辑性

- 题目、选项、答案与反馈文案集中在清晰的数据区。
- CSS 和 JavaScript 均内嵌。
- 默认不依赖外部字体、图片、框架或网络。
- 对内容区和关键逻辑添加必要中文注释。
- 代码结构清晰，教师或开发者可方便修改。
