# 输出模板

## Phase 1 诊断报告

### 1. 原教案结构识别
按教师原有结构逐项概括，不擅自改名。

### 2. 学情分析与学段适配
说明可合理推断的信息，并标记关键假设。

### 3. 教学目标判断
区分知识、技能、策略或素养目标，指出目标与活动是否一致。

### 4. 教学法建议
给出少量、可执行且与原教案兼容的建议。

### 5. 互动潜力评级
使用表格呈现：原教案环节、互动潜力、教学依据、是否建议改造。

### 6. 优先改造建议
推荐 1 至 3 个候选环节，并请教师选择一个。

## Phase 2 活动方案

每个方案使用相同结构：

- 活动名称
- 推荐理由
- 教学目标匹配说明
- 核心互动逻辑
- 课堂组织形式
- 预计时长
- 风险点

## Phase 3 UI Design Specification

- 使用场景
- 推荐视觉风格
- 页面结构
- 色彩与视觉氛围
- 核心组件
- 交互方式
- 反馈机制
- 动效要求
- 禁止项
